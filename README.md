NET SCANNER SIMULATOR JC

An interactive scanner simulator built with Vanilla JavaScript.
Simulates network scanning process, detecting active hosts, open ports, closed ports, and inactive address through a modern interface

DEMO
https://josue-dev-0210.github.io/Net-Scanner-JC/

FEATURES

- Network scan simulation
- Active host detection
- Open port detection
- Closed port detection
- Offline device simulation
- Interactive interface
- Real-time visual feedback
- Responsive design

TECHNOLOGIES USED 

- HTML5
- CSS3
- Vanilla JavaScript
